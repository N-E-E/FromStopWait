## hust-cs-network lab2: FromStopWait
### environment:
- `cmake` >= `3.10`
- `gcc` & `g++`(8.4.0 is ok in my environment)

### usage
First, clone the repo:
```bash
git clone 
cd FromStopWait
```
Then build:
```bash
mkdir build && cd build
cmake ..
make
```
Run:
```bash
cd bin
./stop_wait or ./gbn or ...
```
